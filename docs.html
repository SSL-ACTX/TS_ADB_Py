TS_ADB_Py API Documentation

An API for interacting with Android devices via ADB commands.

API Endpoints

GET /adb/cpu

Get CPU usage statistics from the connected Android device.

GET /adb/battery

Get battery usage statistics from the connected Android device.

GET /adb/device_info

Get device information from the connected Android device.

GET /adb/memory

Get memory usage statistics from the connected Android device.

GET /adb/packages

Get a list of installed packages from the connected Android device.

GET /adb/model

Get the device model and Android version.

GET /adb/network

Get network information from the connected Android device.

GET /adb/storage

Get storage information from the connected Android device.

GET /adb/logcat

Get the logcat output from the connected Android device.

GET /adb/sensors

Get sensor information from the connected Android device.

GET /adb/running_apps

Get a list of currently running apps from the connected Android device.

GET /adb/version

Get the ADB version.

GET /adb/services

Get a list of running services from the connected Android device.

GET /adb/wifi_status

Get the Wi-Fi connection status of the connected Android device.

GET /adb/connected_devices

Get a list of connected USB devices.

GET /adb/packages_info

Get detailed information about all installed packages.

GET /adb/uptime

Get the uptime of the connected Android device.

GET /adb/app_permissions/package_name

Get permissions granted to a specific app.

GET /adb/screenshot

Take a screenshot of the device and return the image data in base64.

POST /adb/reboot

Reboot the connected Android device.

POST /adb/logcat_clear

Clear the logcat buffer.

GET /adb/bt_status

Get Bluetooth status information.

GET /adb/location

Get device location services status.

GET /adb/installed_apps

Get a list of all installed apps with details.

POST /adb/set_property/property_name/value

Set a system property on the connected Android device.

GET /adb/usage_stats

Get app usage statistics from the connected Android device.

GET /adb/brightness

Get the current screen brightness level.

POST /adb/set_brightness/int:value

Set the screen brightness level (0-255).

GET /adb/airplane_mode

Get the airplane mode status of the connected Android device.

POST /adb/set_airplane_mode/int:value

Enable or disable airplane mode on the connected Android device.

GET /adb/notifications

Get the current notifications on the connected Android device.

POST /adb/kill_app/package_name

Kill a running app by its package name.

POST /adb/install_app

Install an APK from the local machine to the connected Android device.

Body: { "apk_path": "path/to/apk" }

POST /adb/uninstall_app/package_name

Uninstall an app by its package name.

GET /adb/media_volume

Get the current media volume level on the connected Android device.

POST /adb/set_media_volume/int:value

Set the media volume level (0-15) on the connected Android device.

GET /adb/foreground_activity

Get the currently foreground app on the connected Android device.

POST /adb/clear_app_data/package_name

Clear data for a specific app by package name.

POST /adb/clear_cache

Clear the cache for all apps on the connected Android device.

GET /adb/device_ip

Get the device's IP address.

GET /adb/system_time

Get the current system time on the connected Android device.

GET /adb/net_info

Get the current network information from the connected Android device.
